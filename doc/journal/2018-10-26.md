## 2018-10-26 

## New feature #utils: #waitForDeepProperty

```
import {waitForDeepProperty} from "utils"

var o = {
  
}

waitForDeepProperty(o, "foo.bar").then(result => {
  lively.notify("Yeah! " + result)
})

lively.sleep(1000).then(() => {
  o.foo = {}
})

lively.sleep(3000).then(() => {
  o.foo.bar = 7
})

```
